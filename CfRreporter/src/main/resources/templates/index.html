Cloud Foundry Application Reporter

CfReporter contacts the cloud controller getting a list of all deployed in all spaces/orgs for which the user has permissions to view.

Usage:

View report by browsing to http://[cf reporter url]/report

Configuration:

Running Locally

To run locally, provide the username, password and cloud foundry url's as environment variables;

vcap.services.credentails.pcf.cfusername

vcap.services.credentials.pcf.cfpassword

vcap.services.credentials.pcf.cftarget

Running as a Cloud Foundry Application

Register the target cloud foundry as a user provided service called 'pcf' and bind this deployed application to it

cf cups pcf -p '{"cfusername":"[user]","cfpassword":"[pass]","cftarget":"https://api.[cf target]"}'

cf push cfreporter -m 512M -p cfreporter.jar --no-start

cf bind-service cfreporter pcf

cf start cfreporter
